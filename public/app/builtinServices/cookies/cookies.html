The $cookies and $cookieStore Services.

Notes:

  • Available in the ngCookies module, we need to load angular-cookies.js and inject the module dependency (ngCookies).
  • We use $cookies to write or read string cookies. $cookies.favorite = $scope.favorite;
  • We use $cookieStore to write or read object cookies.$cookieStore.put('last', $scope.lastBook)
