Service Decorators.

Notes:

  • Service decorators modify the behavior of existing services.
  • Sometimes we can find the expression "wrappers" as synonymous to decorators.
  • We use the decorator function to create the decorator services in the module config: app.config(function ($provide) { $provide.decorator('serviceDecorated', decoratorName) }); function decoratorName ($delegate) { return { serviceDecorated: serviceDecorated } function serviceDecorated (p1) { $delegate.serviceDecorated(p1.toUpperCase); } }
