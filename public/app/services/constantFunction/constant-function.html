Services created with the Constant Function.

The Constant Function just registers a service to be used by the dependency injection system. It does not call the Factory Function nor the Provider Function.

A service created with the Constant Function can be injected into a module configuration function (aka: app.config())

A service created with the Constant Function cannot be overriden by a decorator service.
