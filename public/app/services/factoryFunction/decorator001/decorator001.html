This decorator directive adds the directive as attribute with the size variable as a value of the directive.

In the script, the controller sets a default value for the size and the directive uses the link function.

In the link function, we use scope.$watch to watch if the user enters a new size value, and if that happens we use element.css('font-size', newValue) to change the font-size css attribute of the element.

If the user enters a new % value, the size of this text will change.
