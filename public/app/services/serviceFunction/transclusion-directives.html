Services Created with the Service Function.

The only difference with the Factory Function is that while the Factory Function uses the Module Pattern, the Service Function uses the Constructor Pattern. It uses this. to define its variables and methods, and it requires the keyword new to be instantiated.

The Service Function may be more recommendable than the Factory Function when we are dealing with cases of inheritance hierarchies. In that case, instantiating a new object using the keyword new will ensure that the object inherits properly from its prototype.

Important: in cases of inheritance as the one demonstrated in the first example, the Service function shows its advantage over the Factory Function. If we use the Factory Function here, we would have an error.
